# 🧮 Kalkulator Sederhana

Proyek ini adalah aplikasi kalkulator sederhana yang dibuat menggunakan Python dan Tkinter.  
Tampilannya dibuat dengan tema *dark minimalis*, dan mendukung operasi dasar seperti penjumlahan, pengurangan, perkalian, dan pembagian.

## 🚀 Fitur
- Tampilan dark minimalis
- Input angka dan operator melalui tombol
- Perhitungan otomatis menggunakan tombol "="
- Tombol "C" untuk menghapus input
- Mendukung operasi dasar: +  -  *  /
- Mendukung angka desimal

## 🛠 Teknologi
- *Python 3*
- *Tkinter* (GUI library bawaan Python)
